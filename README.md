# rest-hapi-demo
A simple demonstration of the [rest-hapi](https://github.com/JKHeadley/rest-hapi) plugin.

## Installation

clone the repo
```
$ git clone https://github.com/JKHeadley/rest-hapi-demo.git
```

seed the models
```
$ ./node_modules/.bin/rest-hapi-cli seed
```

## Running the app
```
$ gulp
```
